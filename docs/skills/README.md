# Skill Catalog

This directory documents repo-specific skills for engineers and coding agents working on the DND Booker runtime.

These are not end-user features. They are execution guides for making safe changes in areas that already have strong constraints:

- the autonomous agent runtime
- the AI tool registry
- the Pretext text-layout rollout
- document mutation and layout re-resolution
